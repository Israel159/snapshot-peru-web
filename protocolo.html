Protocolo de Fotomonitoreo

Estándares mínimos para la instalación y manejo de cámaras trampa

📋 Contenido del Protocolo

  • Diseño de muestreo y selección de sitios
  • Instalación de cámaras trampa
  • Configuración y calibración de equipos
  • Períodos de monitoreo recomendados
  • Manejo de datos y metadatos
  • Identificación de especies
  • Análisis de datos estandarizados
  • Compartir resultados
📥 Descargar Protocolo Completo

🎯 Puntos Clave del Protocolo

Arreglo de Cámaras

Número recomendado de cámaras por sitio de muestreo y distancias entre puntos.

Tiempo de Muestreo

Duración mínima de monitoreo por sitio para obtener datos representativos.

Registro de Metadatos

Información obligatoria que debe acompañar cada foto: fecha, hora, coordenadas, habitat.

Privacidad de Datos

Manejo de coordenadas exactas de especies sensibles y datos de comunidades.
